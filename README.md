This Rock Paper Scissors website is a small interactive game where the user plays against the computer in a classic best-of-one format. The interface is clean and minimal, with three clear buttons for rock, paper, and scissors. When a choice is made, the computer randomly generates its move, compares both options using simple game logic, and instantly displays the result (win, lose, or draw). The project, while being a training exercise, focuses on core JavaScript concepts like conditionals, functions, and random number generation, combined with basic HTML structure and CSS styling to create a responsive and user-friendly experience.
